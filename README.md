## Getting started

1. Add mp-conf-files to your project

Maven:
```xml
		<dependency>
			<groupId>es.eisig</groupId>
			<artifactId>mp-conf-files</artifactId>
			<version>1.0.0-SNAPSHOT</version>
		</dependency>
```

Gradle:
```groovy
repositories {
	mavenCentral()
}

dependencies {
	implementation 'es.eisig:mp-conf-files:1.0.0-SNAPSHOT'
}
```
